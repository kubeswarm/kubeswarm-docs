# Security Policy

This repo contains only documentation source. Security vulnerabilities in the **kubeswarm operator or agent runtime** should be reported to the [kubeswarm repo](https://github.com/kubeswarm/kubeswarm/security/advisories/new).

## Reporting an issue with this site

If you find a vulnerability in the docs site itself (e.g. a dependency with a CVE, a script injection in a code example), please report it privately:

**[Report a vulnerability](https://github.com/kubeswarm/kubeswarm-docs/security/advisories/new)**

You can expect an acknowledgement within **48 hours**.
