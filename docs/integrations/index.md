# kubeswarm Integrations - LLM Providers, MCP Tools and More

kubeswarm connects your Kubernetes agents to external services for LLM inference, MCP tool execution, vector storage and monitoring. All integrations are configured declaratively in YAML.

- [LLM Providers](./llm-providers) - Anthropic, OpenAI, Gemini, Ollama, gRPC plugins
- [Task Queue](./task-queue) - Redis Streams, gRPC plugins
- [MCP Servers](./mcp-servers) - Model Context Protocol tool servers
- [Vector Stores](./vector-stores) - Qdrant, Pinecone, Weaviate
- [Notifications](./notifications) - Slack, webhooks
- [Observability](/observability/overview) - OpenTelemetry, Prometheus
- [Autoscaling](/scaling/autoscaling) - KEDA
- [Artifact Storage](./artifact-storage) - S3, GCS
